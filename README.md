## ðŸ“– Deskripsi

Tugas Akhir ini mengembangkan model klasifikasi biner untuk mengidentifikasi meme dengan indikasi _self-harm_ menggunakan pendekatan _intermediate fusion_ yang menggabungkan CLIP untuk representasi visual dan ELECTRA untuk representasi tekstual. Dataset terdiri dari 2,292 meme yang dianotasi menggunakan _ensemble pseudo-labeling_ dengan GPT-4o dan Qwen-VL-Flash.
